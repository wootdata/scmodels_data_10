---[ WENDIGO ]---

Gauna's first and OC player-model for Sven Co-op 5.0

> Model/Texture/Rigging by Gauna
> Compile by The303

Instructions:
1. Extract "wendigo" folder and move it to steamapps\common\Sven Co-op\svencoop_addon\models\player.
2. Run Sven Coop, go Options, select wendigo as player model and press OK.
3. Enjoy!
